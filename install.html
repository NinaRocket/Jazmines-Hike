Install on iPhone / iPad

Apple doesn’t allow a pop-up install button. Do this instead:

  1. Open this page in Safari (not inside another app’s browser).
  2. Tap the Share icon (square with an up arrow).
  3. Scroll and tap Add to Home Screen.
  4. Name it (or keep the default), then tap Add.

Don’t see an Install button?

No worries—some browsers require using the menu:

  • Android (Chrome): tap ⋮ → Install app / Add to Home screen
  • Desktop (Chrome/Edge): look for an Install icon in the address bar, or use the menu
